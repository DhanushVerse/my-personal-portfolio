Project Description

A dynamic and interactive Quiz Application built using modern web technologies. This project focuses on delivering a smooth user experience where users can test their knowledge through a series of questions with real-time feedback and score calculation.

Technical Features

  • Dynamic Content Rendering: Questions and choices are injected into the UI dynamically using JavaScript arrays.
  • State Management: Tracks user progress, current question index, and real-time score calculation.
  • Validation Logic: Immediate feedback system to validate answers and prevent skipping questions.
  • Responsive Design: Optimized for all screen sizes using CSS Media Queries and Flexbox layout.

Key Learnings

Developed a deep understanding of asynchronous-like UI updates, event delegation, and clean code practices. I focused on making the logic modular so that new questions can be added easily by simply updating the data object.
