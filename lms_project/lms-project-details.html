The System In Action

Technical Stack 🛠️

  • Next.js (Frontend & Backend)
  • MongoDB (Database)
  • Tailwind CSS (Styling)
  • Vercel (Deployment)

Key Features ✨

  1. Secure User Authentication
  2. Instructor Course Upload
  3. Student Progress Tracking
  4. Responsive Design for Mobile

Challenges & Learnings

During the development of the course upload feature, I faced challenges with large file handling. I overcame this by implementing AWS S3 for storage and utilizing serverless functions to manage the upload lifecycle, ensuring scalability and speed.
